# Projet personnel – Dev C#

Mod expérimental développé en C# dans un environnement sandbox.

## Description

Ce projet personnel a été réalisé pour mettre en pratique mes compétences en C#.  
Il montre ma capacité à travailler sur des systèmes, à intégrer et améliorer du code existant, et à créer des fonctionnalités.  

## Mon rôle
- Intégration de scripts existants  
- Adaptation et amélioration de systèmes  
- Correction de bugs et optimisation du code

## Pourquoi ce projet
Ce projet m’a permis d’apprendre le C# et de mieux comprendre le fonctionnement de scripts dans un projet réel.
